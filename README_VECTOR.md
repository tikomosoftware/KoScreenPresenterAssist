# KoScreenPresenterAssist

プレゼンテーションや画面共有時に便利な描画・強調表示ツール

## 概要

KoScreenPresenterAssistは、プレゼンテーションや画面共有中に画面上に直接描画したり、カーソルを強調表示したり、画面の一部を拡大表示できるツールです。

## 主な機能

- **描画モード** - 画面上に自由に線や円を描画
- **強調表示モード** - マウスカーソル周辺を強調表示
- **拡大鏡モード** - マウスカーソル周辺を拡大表示
- **ツールバー** - 直感的な操作が可能なフローティングツールバー

## 🚀 インストール・使用方法

### 📦 パッケージ内容
本パッケージは **.NET同梱版** です。
.NET Runtimeのインストール不要で、そのまま実行できます。

### インストール手順
1. ダウンロードしたZIPファイルを解凍
2. `KoScreenPresenterAssist.exe`をダブルクリックして起動

## 使い方

### 起動
アプリケーションを起動すると、タスクトレイにアイコンが表示され、ツールバーが画面上に表示されます。

### ショートカットキー
- `Ctrl + Alt + P`: 描画モード切替
- `Ctrl + Alt + H`: 強調表示モード切替
- `Ctrl + Alt + Z`: 拡大鏡モード切替
- `Ctrl + Alt + X`: 描画を全消去
- `Esc`: 全モード解除

### ツールバー
ツールバーのボタンで各機能を切り替えられます：
- ✏️ : 描画モード
- 🔦 : 強調表示モード
- 🔍 : 拡大鏡モード
- 🗑️ : 全消去
- ⏹️ : 機能OFF
