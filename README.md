# KoScreenPresenterAssist

プレゼンテーションや画面共有時に便利な描画・強調表示ツール

## 概要

KoScreenPresenterAssistは、プレゼンテーションや画面共有中に画面上に直接描画したり、カーソルを強調表示したり、画面の一部を拡大表示できるツールです。

## 主な機能

- **描画モード** - 画面上に自由に線や円を描画
- **強調表示モード** - マウスカーソル周辺を強調表示
- **拡大鏡モード** - マウスカーソル周辺を拡大表示
- **ツールバー** - 直感的な操作が可能なフローティングツールバー

## システム要件

- Windows 10/11
- .NET 9.0 Runtime

## インストール

### ダウンロード

2種類のバージョンから選択してダウンロードしてください：

#### 1. .NET同梱版（推奨）
- **ファイル名**: `KoScreenPresenterAssist-vX.X.X-standalone-release.zip`
- **サイズ**: 約60-80MB
- **.NET Runtimeのインストール不要**
- そのまま実行できます

#### 2. .NET非同梱版（軽量版）
- **ファイル名**: `KoScreenPresenterAssist-vX.X.X-framework-dependent-release.zip`
- **サイズ**: 約1-2MB
- **.NET 9.0 Runtimeが必要**
- Runtimeがインストールされていない場合は、以下からダウンロードしてください
  - [.NET 9.0 Runtime ダウンロード](https://dotnet.microsoft.com/download/dotnet/9.0)

### インストール手順

1. 上記のいずれかのZIPファイルをダウンロード
2. ZIPファイルを解凍
3. `KoScreenPresenterAssist.exe`をダブルクリックして起動

> **推奨**: 初めての方や.NET環境がない方は、**.NET同梱版**をご利用ください。

## 使い方

### 起動

アプリケーションを起動すると、タスクトレイにアイコンが表示され、ツールバーが画面上に表示されます。

### ショートカットキー

| キー | 機能 |
|------|------|
| `Ctrl + Alt + P` | 描画モード切替 |
| `Ctrl + Alt + H` | 強調表示モード切替 |
| `Ctrl + Alt + Z` | 拡大鏡モード切替 |
| `Ctrl + Alt + X` | 描画を全消去 |
| `Esc` | 全モード解除 |

### 描画モード

描画モード中は以下の操作が可能です：

| 操作 | 機能 |
|------|------|
| マウスドラッグ | フリーハンド描画 |
| `Ctrl + ドラッグ` | 円を描く |
| `Shift + ドラッグ` | 直線を描く |
| `1` キー | 赤色に変更 |
| `2` キー | 青色に変更 |
| `3` キー | 黄色に変更 |
| `4` キー | 緑色に変更 |

### ツールバー

ツールバーのボタンで各機能を切り替えられます：

- ✏️ : 描画モード
- 🔦 : 強調表示モード
- 🔍 : 拡大鏡モード
- 🗑️ : 全消去
- ⏹️ : 機能OFF

ツールバーはドラッグで移動できます。

### タスクトレイメニュー

タスクトレイアイコンを右クリックすると以下のメニューが表示されます：

- **ツールバー表示/非表示** - ツールバーの表示を切り替え
- **ヘルプ** - 使い方の詳細を表示
- **Exit** - アプリケーションを終了

## 注意事項

- 描画した内容はモード切替時に非表示になりますが、再度描画モードにすると再表示されます
- 全消去（🗑️）またはCtrl+Alt+Xで完全に削除されます
- アプリケーション終了時に描画内容は保存されません

## トラブルシューティング

### アプリケーションが起動しない

- .NET 9.0 Runtimeがインストールされているか確認してください
- Windows Defenderやセキュリティソフトがブロックしていないか確認してください

### ホットキーが動作しない

- 他のアプリケーションが同じホットキーを使用していないか確認してください
- 管理者権限で実行してみてください

### 描画が表示されない

- 描画モード（✏️）がアクティブになっているか確認してください
- 他のモードに切り替えている場合は、再度描画モードに戻してください
